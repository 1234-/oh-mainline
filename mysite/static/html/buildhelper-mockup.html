Buildhelper

Prerequisite: Open a terminal on your Debian, Ubuntu, or similar system

1 min

For now, OpenHatch developers all seem to use Debian or Ubuntu. These instructions are targeted at that.

Get the code

$ git clone git://gitorious.org/openhatch/oh-mainline.git
10 mins    

If you get "git: command not found", you might need to run:

$ apt-get install git

and then try the command again.
