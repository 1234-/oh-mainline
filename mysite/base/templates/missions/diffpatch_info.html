One of the most critical steps in contributing to projects is showing other people what changes you have made. The diff program inspects the two things you give it and outputs the differences between them. The output of diff is both human readable (you can look at it and easily see where the changes fit in) and machine readable (you can use the patch program to automatically apply the changes to the original file or unapply the changes to the modified file).
