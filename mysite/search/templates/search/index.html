OpenHatch is an open source involvement engine.

OpenHatch makes it easy to:

  • Find new ways that you can help with open source projects
  • Find mentors or mentees and learn new skills
  • Track and show off all your open source software experience in one place.

By providing us with the usernames and email addresses you use in the open source software community, we automatically compile your contributions to open source. We then use your past contributions to make recommendations on how you could contribute further.

First, we collect open bugs from open source bug trackers across the web and highlight tasks that match your skill set.
